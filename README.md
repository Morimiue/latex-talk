# LaTeX 论文排版技术入门

这是我 2022 年为哈尔滨工业大学（威海）图书馆“i 研堂”的 LaTeX 讲座所准备的讲义，旨在循序渐进地为 LaTeX 新手讲解其基本概念和使用方式。

本项目以[知识共享 署名―相同方式共享 4.0 国际版](https://creativecommons.org/licenses/by-sa/4.0/legalcode.zh-Hans)许可协议发布。

本项目在编写过程中参考了：

- [曾祥东《现代 LaTeX 入门讲座》](https://github.com/stone-zeng/latex-talk)
- [清华大学 TUNA 协会《如何使用 LaTeX 排版论文》](https://github.com/tuna/thulib-latex-talk)
